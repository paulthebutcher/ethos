# Ethos

Solo AI Product Incubator concept brief and documentation site.

## Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:3002`

## Structure

```
ethos/
├── app/
│   ├── globals.css    # Tailwind + custom styles
│   ├── layout.jsx     # Root layout with header
│   └── page.jsx       # Main content
├── tailwind.config.js
└── package.json
```
